# Universal Download File Cli

## Installation

```bash
npm install -g universal-download-cli
```

## Usage

```bash
udf --help
```

### How to download file

```bash
udf download

# Example
udf download
? url
? dir
```

### How to list file

```bash
udf list

# Example
udf list folderName
```

## How to run test

**Prerequisites:**
Installed Node.js and NPM

```bash
cd [app_dir]

npm install

npm run test
```
